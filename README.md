# Raytracer

> Raytracer for CSCI 4471 Computer Graphics class project

## Compiling

Build the executable in `bin/` directory.

```bash
make
```

### Cleaning

```bash
make clean
```

## Usage

After compiling (see above):

```bash
./bin/raytracer.o
```

## Testing

```bash
make test
```

Then look into the `out/` for all of the rendered images.
